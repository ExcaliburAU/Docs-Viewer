## Configuration Settings
### Modifying `GameUserSettings.ini`
Add the following section to configure Excalibur Speed Costumes:
```ini
[ExcaliburSpeed]  
HideIcon=false               ; Show/Hide the buff icon (true = hidden)
DisableRiderProtection=false ; Turn off rider protection (true = disabled)
AirControl=1.0               ; Air movement control. Lower = stops quickly, Higher = keeps drifting.
```
